# RobotOmni
### Robot omnidireccional didáctico basado en ROS 2

RobotOmni es una plataforma robótica omnidireccional diseñada con fines educativos.  
Este proyecto integra diseño mecánico, electrónica y software basado en ROS 2, permitiendo que estudiantes y desarrolladores puedan replicarlo, modificarlo y expandirlo.

---

## 📌 Características

- Plataforma omnidireccional
- Diseño mecánico imprimible en 3D
- Placas electrónicas propias (PCB)
- Control de motores y adquisición de sensores
- Integración completa con ROS 2
- Proyecto totalmente replicable

---

## 🛠️ Diseño Mecánico

Contiene:

- Archivos originales en Fusion 360
- Archivos STL listos para impresión 3D

Permite fabricar completamente el robot o modificar su diseño.

---

## 🔌 Electrónica

Incluye:

- Archivos de diseño de PCB
- Archivos Gerber para fabricación
- Archivos editables para modificaciones
- Esquemáticos del sistema

Permite fabricar y adaptar la electrónica del robot.

---

## 💻 Software

Contiene:

- Control de motores
- Lectura de sensores
- Lógica de control
- Paquetes ROS 2
- Integración completa del sistema

---

## ⚙️ Requisitos

### Hardware
- Impresora 3D
- Componentes electrónicos especificados en la carpeta `/electronica`
- Microcontrolador / SBC compatible
- Motores omnidireccionales

### Software
- ROS 2 (Humble o superior recomendado)
- Python 3
- C++
- Herramientas de compilación de ROS 2 (`colcon`)
